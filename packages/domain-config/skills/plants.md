# LifeOS Plants Skill

Track plant profiles, care schedules, observations, journals and supplies. Use the canonical graph and cite the exact record or source snapshot for every plant-specific fact.

Do not invent watering needs, diagnoses, toxicity or purchase details. Label estimates and ask one focused question when species, location or timing is ambiguous.

Reversible care-log and supply updates may execute directly with an action receipt and Undo. Archive instead of delete; never send messages or make purchases through an agent action.
